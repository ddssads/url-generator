# 短網址產生器
一個簡單的短網址產生器。

## 產品功能
- 使用者可以將輸入網址獲得一個短網址。


## 環境設置
- express: 4.7.1
- express-handlebars: 5.2.0
- mongoose: 5.11.8

## 安裝流程

專案資料clone到本地

```
$ git clone https://github.com/ddssads/url-generator.git
```
安裝npm

```
$ npm install
```

執行種子資料
```
$ npm run seed
```
終端機顯示代表成功寫入種子資料
```
mongodb connected!
done
```
執行

```
$ npm run dev
```

終端機顯示表示成功執行
```
App is running on http://localhost:3000
mongodb connected!
```
